# 🎓 LearnNest — Online Learning Platform

Welcome to **LearnNest**! This is a medium-to-large full-stack project built with **Angular (v19)** and **NestJS (v11)** to help you master both frameworks.

## 🚀 Project Overview

LearnNest is a public-facing online learning platform where:
- **Instructors** can create courses, upload video lessons, manage content, and view student enrollment analytics.
- **Students** can browse/search courses, enroll, track their learning progress, watch video lessons, and leave reviews.

## 🛠️ Technology Stack

- **Frontend:** Angular 19, Angular Material, RxJS, Responsive CSS
- **Backend:** NestJS 11, TypeORM, SQLite (simple local DB), JWT Authentication, Multer (file/video uploads)
- **API Documentation:** Swagger UI (auto-generated)
